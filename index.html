📋 Panduan Setup Google Sheets

Langkah 1: Buka Google Sheets → buat spreadsheet baru
Langkah 2: Di spreadsheet, klik Extensions → Apps Script
Langkah 3: Hapus kode lama, paste kode dari File: google-apps-script.gs (file terpisah)

Langkah 5: Set "Who has access" → Anyone → Deploy
Langkah 6: Copy URL yang muncul → paste di atas → Simpan
